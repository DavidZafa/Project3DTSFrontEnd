# DangerZone Models
DangerZone will have 3 models:
* User 
* Animals 
* News

## User model
* User model will have:
   + username
   + password
   + favorite animals array where users can choose specific animals to track
   + Authentication

## Animal model
* Animal model will have:
    + species
    + picture
    + region
    + endangered level
    + population

## News model
* News model will have:
    + title
    + author
    + date posted
    + post
